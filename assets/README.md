# assets/ — intentionally empty

The HourLog IL source (`hour log/time_tracker_8.html`) ships **no** logo,
illustration, photography, or raster/vector icon files. There is nothing to
copy in, and per brand guidance we do **not** invent a logo.

- **Brand mark:** the active currency glyph (₪) set in a 12px-radius ink square —
  pure CSS + a unicode character, not an image. See `guidelines/brand-mark.card.html`.
- **Iconography:** unicode glyphs rendered in Heebo. See `guidelines/brand-icons.card.html`
  and the Iconography section of `readme.md`.
- **Wordmark:** the business name set in plain Heebo.

If you later receive real brand assets (a logo, an illustration set, custom icons,
or the Heebo `.woff2` binaries for self-hosting), drop them here and wire them into
the relevant tokens/components.
